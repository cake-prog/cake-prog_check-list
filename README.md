# cake-prog_check-list
Checklist for the sorting function on the website https://practicesoftwaretesting.com/

Check-list:
        Usability:
                Отображение:
                      1) Выбранное значение сортировки отображается на кнопке выбора сортировки.
                      2) Поле сортировки удобно для использования
                      3) Значения сортировок понятны пользователю.
                Работа выпадающего списка: 
                      1) Список открывается для клика.
                      2) Повторный клик закрывает список
                      3) Клик вне списка закрывает его
        Адаптивность:
                1) Сортировка корректно отображается на разных разрешениях
                2) Dropdown не выходит за границы экрана
                3) Элементы списка остаются читаемыми
       Надёжность интерфейса
                1) Dropdown не ломается при быстром многократном клике
                2) Интерфейс не зависает при быстром переключении сортировок
                3) Нет ошибок в консоли браузера
